# MyBlockModel

A Unity project based on Udemy course "Complete C# Unity Developer 3D: Learn to Code Making Games" 
by Ben Tristem and Rick Davidson that provides assets for a race car and a bicycle.

Submitted by: Amanda Tustin


## User Stories

The following **required** functionality is complete:

* [X] Create 3D Objects to form a car (Race Car).
* [X] Create asset (Race Cars) to hold the form of a car.
* [X] Create 3D Objects to form a second object (Bicycle).
* [X] Create asset (Bicycles) to hold the form of a bicycle.
